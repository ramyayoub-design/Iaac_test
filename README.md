## About
This is a static website that replicates a professional architectural CV layout and a documentation overview page. It uses only vanilla HTML, CSS, and JavaScript (no frameworks) and is designed to be print-friendly for exporting the CV as an A4 PDF.

## Pages
- `index.html` — landing page
- `cv.html` — printable A4 CV layout
- `docs.html` — documentation overview ("docs to keep")

## How to Use
- Open `index.html` in a browser.
- Use the top navigation to access the CV and Docs pages.
- On the CV page, use the "Print / Save as PDF" button or the browser Print dialog to export the CV as a PDF.
